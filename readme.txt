"Que dice Uruguay" project was released in july 2019 in the context of the uruguayan presidential campain of that year.

The project consist on downloading tweeter data regarding presidencial candidates with the purpose of generarte a 
sentiment analysis of this social media regarding uruguayan politicians.  

The project consist in two blocks: main.py was responsible of downloading, processing and generate predictions, 
which results were stored in a database for be used in a django project for deploying on the web site.
